# Boba Fusion
Watermelon merge/Suika mobile game with Bubble Tea aesthetic developed in Godot 4.3.

This is a upgrade of [Watermelon Fusion](https://github.com/koopmyers/watermelon-fusion) project for Godot 3.5.
